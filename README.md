# Projeto Landing Page TECH

## Descrição
Este projeto é uma landing page para uma startup fictícia chamada "Startup Tech". A página apresenta informações sobre a empresa, seus serviços, depoimentos de clientes, planos de assinatura e um formulário de contato.

## Tecnologias Usadas

### Frontend
- **HTML5**: Estrutura básica do site.
- **CSS3**: Estilização do site.
- **Bootstrap 5.3.3**: Framework CSS para design responsivo.
- **Bootstrap Icons 1.11.3**: Ícones para navegação e conteúdo.
- **Google Fonts**: Fontes para tipografia personalizada.
- **jQuery 3.7.1**: Biblioteca JavaScript para manipulação DOM e interações.
- **jQuery Mask Plugin**: Plugin jQuery para máscaras de entrada.
- **jQuery Validate Plugin**: Plugin jQuery para validação de formulários.
- **Custom CSS**: Estilização personalizada em `style.css`.

### Estrutura do Projeto
- **`index.html`**: Estrutura HTML principal.
- **`style.css`**: Estilos personalizados.
- **Imagens**: Localizadas na pasta `/img`.
- **JavaScript**: Scripts personalizados em `/js/main.js`, além dos plugins jQuery.

## Seções do Site
- **Cabeçalho (Header)**: Barra de navegação com links para diferentes seções do site.
- **Chamada para Ação**: Seção de destaque com título, descrição e botão de ação.
- **Features**: Descrição das principais características e benefícios dos serviços oferecidos.
- **Depoimentos (Feedbacks)**: Comentários e avaliações de clientes satisfeitos.
- **Planos**: Detalhes dos diferentes planos de assinatura oferecidos pela empresa.
- **Contato**: Formulário de contato e informações de contato da empresa.

## Instruções para Executar
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em um navegador web.
